# Chrome Extension Display Switch
非表示の要素を表示するChrome extension。

### Support Property
| property |  |
|:-----------|:------------|
| display | &#x2713; |
| opacity | - |
| visibility | - |
| height 0 | - |

# Getting Started
git clone or Download ZIP

```
git clone https://github.com/konweb/extension-display-switch
```

# Settings
- 設定 > 拡張機能ページに`.crx`ファイルをドラッグ&ドロップ
